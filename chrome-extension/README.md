## ğŸ“– HOW TO USE

### Quick Start
1. Go to any product website (furniture store, supplier, etc.)
2. Click the extension icon â†’ side panel opens
3. Select your **Project â†’ Room â†’ Category â†’ Item**
4. Click **"Smart fill from current page"** - AI extracts product info
5. Review the data, make edits if needed
6. Click **"Link Product to FFE Item"**

### Cascading Selection Flow
The extension uses a step-by-step selection process:
1. **Select Project** â†’ Project becomes a tag, room dropdown appears
2. **Select Room** â†’ Room becomes a tag, category dropdown appears
3. **Select Category** â†’ Category becomes a tag, item list appears
4. **Select or Create Item** â†’ Ready to clip!

Click the **Ã—** on any tag to go back to that step.

### Link to Existing FFE Items
- After selecting a category, you'll see existing FFE items
- Items marked with âš ï¸ need product specs
- Select an item to link the clipped product to it
- Or choose "Create new item" to add a new FFE item

### Manual Text Selection
1. Click the **+** button next to any field
2. Click on text in the webpage
3. The text is captured into that field

### Clipping Images
- **Right-click** any image on the page
- Select **"Clip Image to StudioFlow"**
- Image appears in the extension

### Reorder Images
- Drag images to reorder them
- First image = main product photo in StudioFlow
